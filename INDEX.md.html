happy

happy is a simple TCP happy eyeballs probing tool. It uses non‐blocking connect() calls to establish connections concurrently to a number of possible endpoints. This tool is particularly useful to determine whether happy eyeball [RFC 6555] applications will use IPv4 or IPv6 if both are available.
